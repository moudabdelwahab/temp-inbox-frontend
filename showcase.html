🎯 نظرة عامة على المشروع

Temp Inbox هو تطبيق ويب احترافي يسمح للمستخدمين بالحصول على عناوين بريد إلكترونية مؤقتة فوريًا دون الحاجة إلى إنشاء حساب. يتم توليد عنوان بريد فريد لكل جلسة باستخدام Gmail Plus Addressing (user+RANDOM_ID@gmail.com)، ويتم عرض الإيميلات الواردة في الوقت الفعلي مع تأخير أقصى 10 ثوان.

المشروع مبني بـ HTML + CSS + JavaScript فقط (بدون frameworks) مما يجعله سريع جدًا وخفيف الوزن. يتم الربط مع Supabase للقراءة فقط، بينما يتولى backend منفصل مسؤولية جلب الإيميلات من Gmail API.

🎯 الميزات الرئيسية المُنفذة

✅ إدارة الجلسات

توليد معرّف جلسة فريد (UUID) تلقائيًا عند تحميل الصفحة، مع تخزينه في localStorage مع انتهاء الصلاحية بعد 24 ساعة. يمكن توليد جلسة جديدة بضغطة زر واحدة.

✅ توليد الإيميلات

توليد عنوان بريد مؤقت باستخدام Gmail Plus Addressing: user+SESSION_ID@gmail.com. يتم عرض الإيميل بوضوح مع أزرار نسخ وتحديث.

✅ جلب الإيميلات

جلب الإيميلات من Supabase كل 5-10 ثوان (Polling)، مع دعم Real-time Subscriptions كـ primary method. الاستقصاء يتكيف تلقائيًا ويتوقف عند عدم وجود تحديثات جديدة.

✅ عرض محتوى الإيميل

عرض محتوى الإيميل الكامل مع تنظيف HTML باستخدام DOMPurify لمنع هجمات XSS. يتم عرض HTML إن وجد، وإلا يتم عرض النص العادي.

✅ الأمان

استخدام DOMPurify لتنظيف محتوى HTML، عدم استخدام dangerouslySetInnerHTML، وتجنب جميع ثغرات XSS الشائعة.

✅ التصميم المستجيب

دعم كامل للأجهزة المحمولة والأجهزة اللوحية والحواسيب، مع تصميم احترافي وحديث يتضمن Dark Mode.

🎁 الفوائد الرئيسية

📊 استكشاف البيانات بطريقة أكثر بديهية

الواجهة الأمامية توفر طريقة سهلة وبديهية لاستكشاف الإيميلات المستقبلة، مع عرض قائمة الإيميلات وتفاصيلها بشكل منظم وسهل الفهم. يمكن البحث والتصفية بسهولة.

📈 فهم الاتجاهات بشكل أفضل

من خلال عرض الإيميلات بترتيب زمني وتجميعها حسب المرسل والموضوع، يمكن فهم أنماط الإيميلات الواردة والاتجاهات بشكل أفضل. الإحصائيات المرئية توساعد في اتخاذ قرارات أفضل.

💾 سهولة الحفظ والمشاركة

يمكن نسخ عنوان البريد المؤقت بضغطة زر واحدة ومشاركته مع الآخرين. الجلسة تُحفظ تلقائيًا في localStorage، مما يسمح بالعودة إلى نفس الإيميل لاحقًا.

المتطلبات

التثبيت

# 1. استنساخ المشروع
git clone https://github.com/moudabdelwahab/temp-inbox-frontend.git
cd temp-inbox-frontend

# 2. تحديث ملف config.js بمفاتيح Supabase
# قم بتحرير js/config.js وأضف:
# SUPABASE_URL: 'https://your-project.supabase.co'
# SUPABASE_KEY: 'your-anon-key'

# 3. تشغيل خادم محلي
python -m http.server 8000
# أو
npx http-server

# 4. الوصول للتطبيق
# افتح http://localhost:8000 في المتصفح
